Pixla

Visual-centric social media

Utilizes Node.js with Express-Handlebars for routing and templating, Bulma.io CDN, Okta Authorization, and the Pexels image API. Linked to an SQL Database; deployed to Heroku with the JawsDB plugin.

Wordcloud

Cloud based story writing

Utilizes BigHugeThesaurus and Pexels APIs for image and word search. Elements of Bootstrap and Materialize CDN. Linked to a Firebase Real-time Database & Authorization. Also makes use of Johann Burkard's Highlight plugin.

Booktionary

React book search app

Full-stack MERN application utilizing React and React-Bootstrap components in conjunction with a MongoDB database. Uses Axios for React-dom-router requests. Deployed to Heroku; utilizes the Heroku mLab plugin.
