About me

I am a graphic & web designer with clients ranging from the architectural sector to the United States Census Office. I specialize in graphic & layout design, product photography, copy, video, and illustration - but also have experience with 3D, animation, sound design, and much more. In college, I worked with every material from data to concrete, from drawing and painting to film. I enjoy tinkering with the details as well as painting a big picture; I've been tapped for singular projects like logos, but have also acted as the sole artist at a multinational instrument case manufacturing company.

Put simply, if you need something created, I can create it. Contact me so we can get started on your project, or crafting your comprehensive brand identity.

Wordcloud

Writing app with features meant to stir the imagination and help maintain flow. Word and image search conducted with BigHugeThesaurus and Pexels APIs. Elements of Bootstrap and Materialize CDN. Linked to Firebase Real-Time Database & Auth for automatic saving if logged in. Also makes use of Johann Burkard's Highlight plugin.

Pixla

Visual social media site linked to an SQL database for image sharing & commenting. Utilizes Node.js with Express-Handlebars for routing and templating, Bulma.io CDN, Okta Authorization, and the Pexels image API. Deployed to Heroku with the JawsDB plugin for SQL.

Github Repository

Booktionary

Book searching and saving app, pulling from the Google Books API as a resource. Full-Stack MERN application utilizing React and React-Bootstrap components in conjunction with a MongoDB database. Uses Axios for React Router requests. Deployed to Heroku with the mLab plugin for MongoDB.

Github Repository
